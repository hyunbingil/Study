# 2. 비밀번호 조건 달기
: 정규표현식과 .length를 이용해서 조건을 달 수 있다.
- 숫자와 문자를 포함해서 만들어라.
``` js
var pattern1 = /[0-9]/;
var pattern2 = /[a-zA-Z]/;
```
- 비밀번호는 8자리 이상.
``` js
if(!pattern1.test(pw)||!pattern2.test(pw)||pw.length<8||pwlength>50){
    alert("영문+숫자 8자리 이상으로 구성하여야 합니다.");
    return false;
}   
```
