### justify-content
: ```justify-content: ~~~```
1. flex-start\
: 요소들을 컨테이너의 왼쪽으로 정렬한다.
2. flex-end\
: 요소들을 컨테이너의 오른쪽으로 정렬한다.
3. center\
: 요소들을 컨테이너의 가운데로 정렬한다.
4. space-between\
: 요소들 사이에 동일한 간격을 둔다.
5. space-around\
: 요소들 주위에 동일한 간격을 둔다.

### align-items
: ```align-items: ~~~```
1. flex-start\
: 요소들을 컨테이너의 꼭대기로 정렬한다.
2. flex-end\
: 요소들을 컨테이너의 바닥으로 정렬한다.
3. center\
: 요소들을 컨테이너의 세로선 상의 가운데로 정렬한다.
4. baseline\
: 요소들을 컨테이너의 시작 위치에 정렬한다.
5. stretch\
: 요소들을 컨테이너에 맞도록 늘린다.
