## 박스(Box) 모델
: HTML의 모든 요소는 Box의 형태를 갖는다.
- 장점\
: 컨텐츠를 테트리스처럼 조밀하게 배치할 수 있다.\
: 박스의 크기를 계싼해 쉽게 브라우저로 출력할 수 있다.
### box 모델의 구성
: 박스 형태 전체를 가리킨다.
#### 1. 내용 (Content)
: 이미지나 텍스트와 같이 우리가 코드 상에서 태그 사이에 담은 내용, __실제로 담고 있는 부분__
#### 2. 패딩(Padding)
: 컨텐츠와 경계선 사이의 여백
#### 3. 경계선(Border)
: 말 그래도 컨텐츠를 감싹 있는 테두리
#### 4. 마진(Margin)
: 경계선 밖의 여백
